Руководство программиста (Храмов М. ИПО-14Б)

Try hard

Проект Try Hard является игрой в стиле старых космо - платформеров

Играя в игру игрок погружается в воспоминания о начале игровой индустрии.

Сценарий использования 1 - Загрузка игры

Пользователь нажимает кнопку Fire и начинается игра.

Используются глобальные переменные и функция инициализации игры.

function fire() {
    if (inPlay == 0) {
        if (selec == 1) {
            selec = 9;
            inPlay = 1;
            init();
        }
    } else {
        if (missileY == 99) {
            missileY = 18;
            missileX = myChrx + 3;
        }
    }
}

Сценарий использования 2 - Управление

Пользователь нажимает кнопку Fire для стрельбы и кнопки Left Right для перемещения по полю.

Применяются стили в зависимости от выбора пользователя. Берется текущая дата.

function starter() {
    document.fmark.win.value = cr + cr + cr + cr + cr 
        + cr + cr + "Are you ready?" + cr + cr + cr + "Click FIRE!";
    selec = 1;
}

/**
* Функция для перемещения влево
*
* Посимвольное перемещение влево
*
*/
function goLeft() {
    myMove = 0;
    myChrx = myChrx - 1;
    
    if (myChrx <= 0) {
        myChrx = 1;
    }
}

/**
* Функция для перемещения вправо
*
* Посимвольное перемешение вправо
*
*/
function goRight() {
    myMove = 0;
    myChrx = myChrx + 1;
    
    if (myChrx >= 56) {
        myChrx = 55;
    }
}

Сценарий использования 3 - Рестарт

Пользователь нажимает кнопку Reset для сброса игрового цикла.

Сбрасывает значения глобальных переменных к первоначальному.

function restarter() {
    inPlay = 0;
    blankNo = 0;
    totalBlank = "";
    lr = 1;
    lowerLf = "";
    downLf = "";
    downNo = 0;
    missileX = 99;
    missileY = 99;
    myChrx = 1;
    myBlank = "";
    myMove = 0;
    selec = 1;
    starter();
}
